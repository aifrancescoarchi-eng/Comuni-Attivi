## NOTE TECNICHE PER DESIGNER

### Tono generale
- Diretto, professionale, concreto
- Zero marketing aggressivo
- Linguaggio che parla a sindaci e segretari
- Zero fronzoli, solo contenuto utile

### Palette colori suggerita
- Primario: Blu scuro (#1e3a5f o simile)
- Secondario: Grigio scuro (#2d3748)
- Sfondo: Bianco/grigio chiarissimo (#f7fafc)
- Accento: Verde tenue (#38a169) o arancio tenue (#ed8936)
- Testo: Nero/Grigio scuro (#1a202c)

### Tipografia
- Font: Inter, Roboto o system sans-serif
- Gerarchia chiara: H1 grande, H2 medio, body leggibile

### Layout
- Mobile-first
- Hero full-width
- Sezioni con padding generoso
- Card semplici, bordi sottili o ombra leggera
- Nessun parallax o animazioni pesanti

### Requisiti tecnici
- Responsive (mobile perfetto)
- Peso pagina < 1MB
- Accessibilità: contrasto buono, alt text
- SEO: title, description, heading corretti
